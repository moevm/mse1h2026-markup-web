### Запланированные задачи
- Реализован макет web-интерфейса
- Реализовано API для разметки изображения при помощи заранее обученной модели и дообучения модели
- Реализован сервер для отображения макета web-страницы
- Был найден датасет, при помощи которого модель была протестирован
### Выполненные задачи
Все задачи, поставленные в рамках текущей итерации были выполнены
